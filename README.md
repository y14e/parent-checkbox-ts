# parent-checkbox.ts
WAI-ARIA compliant [checkbox](https://www.w3.org/WAI/ARIA/apg/patterns/checkbox/) (mixed-state) pattern implementation in TS.
## Installation
```ts
import ParentCheckbox from './parent-checkbox';

const myParentCheckbox = new ParentCheckbox(...);
```
